# SICAP PNRR - proiect evidence.dev

## Dezvolta proiectul local

Executa comenzile de mai jos pentru a rula proiectul local

```bash
git clone https://github.com/ciocan/sicap-pnrr.git
cd sicap-pnrr
npm install
npm run sources
npm run dev -- --host 0.0.0.0
```
